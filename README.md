##        Editor de Pixel Art con Electron JS y Vanilla JS

### Ejecutar el programa
#### **1. Clonar el Repo**
> **git clone https://github.com/marioxe301/PXEditor.git**
#### **2. Instalar las Dependencias**
> **cd PXEditor** <br/>
> **npm install**
#### **3.Ejecutarlo**
> **npm start**
